## Interview Talk Track

```text
This started as an information consulting project about LVMH's grey-market exposure. I wanted to go beyond a recommendation deck, so I built a working prototype that shows how an intelligence system could detect suspicious listings, score distributor risk, explain evidence, and model allocation pressure.

The most important design decision was explainability. Instead of showing black-box risk scores, every escalation is tied back to visible evidence such as price deviation, platform risk, seller behavior, regional exposure, and distributor signals.

The project is synthetic because I did not have access to proprietary LVMH data, but the architecture is realistic: Flask API, SQLAlchemy data model, ML scoring layer, dashboard UI, and a deployed Render web service.
```
